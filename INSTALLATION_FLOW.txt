  ┌─────────────────────────────────────────┐

  ┌─────────────────────────────────────────┐

═══════════════════════════════════════════════════════════════════════
                     USER INSTALLATION FLOW
═══════════════════════════════════════════════════════════════════════

1. USER OPENS DISCORD
   │
   ├─ Opens Revenge/Vendetta Settings
   │
   └─ Goes to "Plugins" section
       │
       ▼

2. CLICKS "INSTALL" or "+"
   │
   ├─ Sees "Install Plugin" dialog
   │
   └─ Input field for URL
       │
       ▼

3. PASTES YOUR GITHUB URL
   │
   ├─ github.com/YOUR_USERNAME/local-message-editor
   │
   └─ Clicks "Install"
       │
       ▼

4. REVENGE/VENDETTA DOWNLOADS
   │
   ├─ Fetches manifest.json
   ├─ Reads "main": "src/index.tsx"
   ├─ Downloads src/index.tsx
   │
   └─ Validates plugin
       │
       ▼

5. PLUGIN APPEARS IN LIST
   │
   ├─ "Local Message Editor" shown
   ├─ Toggle to enable
   │
   └─ Plugin loads
       │
       ▼

6. PLUGIN ACTIVE! ✅
   │
   ├─ Console: "[LocalMessageEditor] Plugin loaded successfully!"
   ├─ Long-press message shows "📝 Edit Locally"
   │
   └─ User can edit messages!
